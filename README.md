## 开发

```bash
# 克隆项目
git clone https://github.com/TickNet-Hnust/labor-education-admin

# 进入项目目录
cd labor-education-admin

# 安装依赖
pnpm i
# 建议使用16版本的node环境，目前18版本node环境暂不稳定
```

## 运行

~~~bash
# 启动服务
pnpm dev
~~~
