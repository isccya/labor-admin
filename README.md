## 开发

```bash
# 克隆项目
git clone git@github.com:isccya/labor-education-admin.git

# 进入项目目录
cd labor-education-admin

# 安装依赖
pnpm i
# 建议使用16版本的node环境，目前18版本node环境暂不稳定
```

## 运行

~~~bash
# 启动服务
pnpm dev
~~~

## 大致介绍

项目框架使用[ RuoYi 权限后台管理系统 ](http://doc.ruoyi.vip/ruoyi/) ,这个版本引入了windicss,并修改了样式bug
